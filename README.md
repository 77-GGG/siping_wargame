# 解放战争·东北篇 —— 四平保卫战

### 作者：史纲汇报第12组、Claude Code

基于**杀戮尖塔（Slay the Spire）** 战斗机制，以**毛泽东军事思想**为理论依据的 roguelike 卡牌游戏。

## 快速开始

```bash
# 安装依赖（GUI版需要pygame）
pip install -r requirements.txt

# 终端文字版
python main.py

# GUI图形界面版
python gui_main.py
```

## 游戏模式

### 手动战斗模式

玩家选择指挥官，亲自操控出牌，指挥四平保卫战。

- 每回合获得 3 点能量（群众基础），抽 5 张牌
- 打出攻击牌、技能牌消耗能量，击退敌军
- 能力牌打出后永久生效（如"土地改革"每回合+1能量）
- 结束回合后敌方行动，随后进入下一回合

### 史实自动演示模式

按 1946 年四平保卫战的历史进程自动出牌，每回合附带历史叙事，按回车逐步推进，类似翻页演示。

## 指挥官

| 指挥官 | 风格 | 阵地/兵力/指挥 | 特点 |
|--------|------|---------------|------|
| 雷霆 | 先锋型 | 40/50/20 | 灵活度2 主动权3(暴击15%) 首回合爆发 |
| 磐石 | 基石型 | 60/50/25 | 灵活度1 主动权1(暴击5%) 每回合+3格挡 |

## 核心机制

### 三层血条（战略纵深）

伤害依次穿透：**阵地 → 兵力 → 指挥**。指挥归零则战败。

- 阵地击穿 → debuff"失去阵地"（灵活度-1，能量-1）
- 兵力击穿 → debuff"兵力枯竭"（伤害减半）

### 资源系统

| 资源 | 说明 |
|------|------|
| 群众基础（能量） | 每回合用于打出卡牌 |
| 灵活度 | 消耗后可转换手牌或置顶牌 |
| 战略主动权 | 每点=5%暴击率（上限25%），暴击伤害×1.5 |

### 四平保卫战（Boss战三阶段）

1. **运动防御**（回合1-4）：两个敌人，运动战加成，大洼反击窗口
2. **城防对峙**（回合5-9）：防线系统（40点），敌军持续增兵，南线牵制
3. **塔子山危机**（回合10+）：新六军Boss登场，塔子山要塞20点，陷落则防线崩溃

第8回合起可选择主动撤退，越晚撤奖励越高但风险越大。

## 卡牌一览

共 **34张** 卡牌：攻击牌11张 / 技能牌11张 / 能力牌8张 / 战役专属4张

代表性卡牌：

| 名称 | 类型 | 费用 | 效果 | 理论依据 |
|------|------|------|------|---------|
| 集中兵力 | 攻击 | 1 | 8伤害，已出攻击牌则+5 | "集中绝对优势兵力，各个击破" |
| 诱敌深入 | 技能 | 1 | 本回合受伤-75%，下回合首攻翻倍 | "主动放弃阵地引诱敌军" |
| 土地改革 | 能力 | 2 | 永久每回合+1能量 | "速在全满解决土地问题" |
| 化四平为马德里 | 技能 | 3 | 本回合不死+攻击+50%，回合末-10HP | "死守四平，寸土必争" |

## GUI版设置

GUI版支持分辨率和UI缩放调整，在标题画面点击"设置"：

- **分辨率**：1200×800 / 1440×960 / 1600×1000 / 1920×1080 / 2560×1440 / 全屏
- **UI缩放**：100% ~ 200%

推荐搭配高分辨率+较大缩放以获得最清晰的文字显示。
