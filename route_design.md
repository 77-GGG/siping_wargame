## 第一层：立足东北（1945.11 — 1946.2）

玩家刚刚"出关"，部队远道而来，疲惫且地形不熟。
这一层的战斗小而精，主要让玩家熟悉卡牌系统，积累初始资源。

### 节点1A：山海关保卫战

- **时间**：1945年11月16日
- **史实**：李运昌部1.3万人守卫山海关，阻击杜聿明第13军、第52军。我军兵力单薄，激战一天后山海关失守，但为主力集结锦州争取了时间。
- **敌人**：国民党第13军先头团（普通怪，HP30/攻5）
- **特点**：教学关卡。敌人装备一般，但我军刚到东北，初始手牌少。
- **胜利条件**：坚守3回合（防御教学）或消灭敌前锋
- **失败后果**：无惩罚（剧情战斗），直接进入下一节点
- **奖励**：基础攻击牌×1、"破坏交通"技能牌
- **叙事**：*"八路出关，十万大军与两万干部涌入白山黑水。山海关是进入东北的第一道门。"*

### 节点1B：秀水河子战斗

- **时间**：1946年2月13日
- **史实**：东北民主联军在辽宁法库秀水河子伏击国民党第13军第89师一个加强营及保安团，歼敌近千人。这是我军进入东北后第一场歼灭战胜利，极大鼓舞了士气。
- **敌人**：第89师加强营（HP35/攻6）+ 保安团（HP20/攻3）
- **特点**：伏击战。首回合玩家先手且有"运动战加成"buff（攻+3持续2回合）
- **胜利条件**：消灭全部敌人
- **奖励**：战略主动权+1、可选获得"夜战突袭"或"侧翼包抄"
- **叙事**：*"出关以来首胜！东北能打仗，东北能打胜仗！梁兴初山东一师在秀水河子设伏，全歼敌一个营。"*

### 节点1C：沙岭战斗

- **时间**：1946年2月19日
- **史实**：新四军第3师在辽中沙岭阻击新6军，战斗惨烈，我军伤亡2000余人，虽未达成歼敌目标，但摸清了国军美械部队的战法，积累了宝贵经验。
- **敌人**：新六军先头营（精英怪，HP40/攻8，特性："美械装备"——首次受伤减半）
- **特点**：硬仗。敌人攻高防高，考验防御牌运用。战斗结束后无论胜败都会给出"教训"buff。
- **胜利条件**：消灭敌人或坚守4回合
- **奖励**：
  - 胜利：获得"整军经武"技能牌 + 灵活度+1
  - 坚守满回合：获得"养精蓄锐"技能牌
- **叙事**：*"新四军3师的老兵第一次遭遇全副美械的新六军。血的教训换来宝贵经验——以后对付五大主力，不能硬碰硬。"*

**路线分叉**：第一层三个节点中，玩家必须经过1A（教学），然后在1B和1C中**至少选择一个**（可以两个都打）。多打一场意味着更多卡牌但也更多消耗。

## 第二层：根据地建设与南满拉锯（1946.1 — 1946.3）

这一层战斗规模增大，敌人更强，但玩家也开始获得能力牌。
三条路线对应三个战略方向，互相独立，玩家选择一条主线+可选一条支线。

### 节点2A：鞍海战役（南满攻势）

- **时间**：1946年1月5日
- **史实**：东北民主联军南满部队攻占鞍山、营口、海城等南满重要城市，歼灭伪军1.2万人。南满一度形势大好。
- **敌人**：伪军守备队×2（HP25/攻4）+ 保安团（HP20/攻3）
- **特点**：多敌战斗，考验AOE和"各个击破"。敌人弱但数量多。
- **胜利条件**：消灭全部敌人
- **奖励**：金币（群众基础）+20%、可选获得"南线牵制"攻击牌或"土地改革"能力牌
- **叙事**：*"南满攻势！鞍山、营口、海城相继解放。'歼敌一两个师即可牵制大量顽军不得北进'——南线牵制，北线主攻。"*

### 节点2B：通辽-郑家屯剿匪

- **时间**：1946年2月
- **史实**：西满军区在通辽、郑家屯等地剿灭大批土匪武装，肃清后方，为根据地建设创造条件。全区累计歼匪7万余人。
- **敌人**：土匪头子（HP35/攻5，特性："散兵游勇"——每回合召唤1个小喽啰HP10/攻2）+ 小喽啰×2
- **特点**：持续刷怪战，考验持久作战能力。需要先击杀头目才能阻止刷新。
- **胜利条件**：消灭土匪头子
- **奖励**：可选获得"建立根据地"能力牌或"人民战争"能力牌
- **叙事**：*"'将正规军分散到各军分区，发动群众、消灭土匪、建立政权。'根据地的雏形，从剿匪开始。"*

### 节点2C：本溪防御战

- **时间**：1946年3月—4月
- **史实**：南满我军在本溪地区依托山地两度击退国军第25师、第14师的进攻，歼敌4000余人，成功牵制了南线约8万敌军不得北进，为四平保卫战赢得战略空间。
- **敌人**：第25师攻击群（HP45/攻7）+ 第14师侧翼（HP35/攻6）
- **特点**：防御战，地形加成——"山地防御"buff（所有格挡+30%）。敌人每3回合会增援。
- **胜利条件**：坚守5回合且至少消灭1个敌人
- **奖励**：战略主动权+1、可选"运动防御"升级版或"力战不退"攻击牌
- **叙事**：*"本溪是南满的钉子。只要本溪不倒，廖耀湘的8万大军就被钉死在南线。每多守一天，四平就多一分胜算。"*

**路线规则**：三个节点选2个。2A和2C是军事路线，2B偏向建设路线。

## 第三层：兵临四平（1946.3 — 1946.4）

四平保卫战的直接前奏。战斗规模进一步提升，这里的表现直接影响Boss战的初始条件。

### 节点3A：一战四平（四平攻城战）

- **时间**：1946年3月15日—17日
- **史实**：苏军撤离后，钟伟、马仁兴指挥6000余人从三面攻击四平，2小时结束战斗，歼灭国民党守军及伪军3000余人，活捉辽北省主席刘翰东。
- **敌人**：伪军守城部队（HP30/攻4）+ 保安司令部（HP25/攻3，特性："一触即溃"——HP降至50%以下时攻击归0）
- **特点**：攻城战，节奏快。首回合有"奇袭"buff（首张攻击牌伤害×2）。
- **胜利条件**：4回合内消灭全部敌人
- **奖励**：
  - 4回合内胜利：获得"奇袭夺城"攻击牌升级版 + Boss战额外起始能量+1
  - 普通胜利：获得基础卡牌选择
- **叙事**：*"3月13日苏军撤出四平，3月17日凌晨2点战斗打响，中午即告结束。一战四平，旗开得胜！活捉伪省主席刘翰东。"*

### 节点3B：长春解放战

- **时间**：1946年4月14日—18日
- **史实**：杨国夫率山东第7师配合其他部队攻克长春，歼灭守军2万余人。长春解放后，杨国夫部火速南下增援四平。同期哈尔滨、齐齐哈尔也相继解放。
- **敌人**：长春守军总部（HP50/攻6）+ 伪满残余（HP25/攻4）+ 保安旅（HP30/攻5）
- **特点**：多目标攻坚战。三个敌人互相提供增益——"协防"：每存活一个友军攻+1。
- **胜利条件**：消灭长春守军总部
- **奖励**：
  - 全歼：战略主动权+2 + "招募志愿兵"技能牌（长春民众踊跃参军）
  - 仅击杀总部：战略主动权+1
- **叙事**：*"长春、哈尔滨、齐齐哈尔——北满三大城市相继解放！'只要我们有了东北，全国革命就有了巩固的基础。'"*

**路线规则**：两个节点**必须都打**。顺序自选：
- 先打3A再打3B → Boss战初始手牌+1（先拿下四平，再北上席卷）
- 先打3B再打3A → Boss战初始格挡+10（先稳定后方，再南下据守）

## Boss战：四平保卫战（1946.4.18 — 5.19）

（已实现，见 `scenarios.py` 中的 `SipingBattle`）

### Boss战初始条件受路线影响

| 路线成就 | 对Boss战的加成 |
|---------|-------------|
| 秀水河子全歼 | 战略主动权+1（开局暴击率更高） |
| 沙岭战斗胜利 | 获得"整军经武"永久buff（知己知彼） |
| 本溪防御战胜利 | 阶段二延迟1回合（南线牵制成功） |
| 鞍海战役全歼 | 阶段二敌人攻-2（南满威慑） |
| 通辽剿匪成功 | 每回合额外+1群众基础（根据地建设成效） |
| 一战四平速胜 | 起始能量+1（准备充分） |
| 长春全歼 | 获得"招募志愿兵"（兵员补充） |

## 路线选择示例

**路线A（进攻型·雷霆推荐）**：
1A山海关 → 1B秀水河子 → 2A鞍海战役 → 2C本溪防御战 → 3A一战四平 → 3B长春 → Boss

**路线B（防御型·磐石推荐）**：
1A山海关 → 1C沙岭战斗 → 2B通辽剿匪 → 2C本溪防御战 → 3B长春 → 3A一战四平 → Boss

**路线C（全面型·高风险高回报）**：
1A山海关 → 1B秀水河子 → 1C沙岭 → 2A鞍海 → 2B通辽 → 3A一战四平 → 3B长春 → Boss

## 设计理念

1. **历史即关卡**：每个节点都是真实战斗，叙事文字直接引用毛泽东军事文集原文
2. **选择即战略**：路线分叉映射"集中兵力"原则——打更多关卡获得更多牌，但也承受更多消耗
3. **前因后果**：前期战斗的结果直接影响Boss战难度，体现"统筹全局"思想
4. **城市vs农村**：2A/2C走城市军事路线，2B走农村根据地路线，呼应策划案中的"城市路线vs农村路线"设计
5. **南线牵制**：2A和2C的本溪/鞍海战斗直接影响Boss战中新六军的出场时间，还原"南线牵制北线"的真实战略逻辑
